array

KICK.math.vec3 array ( count , ref )
Create a continuous array in memory mapped to vec4. Example 
var ref = {};
var v = KICK.math.vec4.array(2,ref);
v[1][1] = 1;
ref.mem[5] == v[1][1];
Will be layed out like this: 
[vec4][vec4] = [0][1][2][3][4][5][6][7]
Parameters:
count <Number> Number of vec 3 to be layed out in memory
ref <Object> Optional, if set a memory reference is set to ref.mem
Returns: KICK.math.vec3
New vec3

wrapArray

Array[KICK.math.vec4] wrapArray ( array )
Wraps a Float32Array with multiple vec4 arrays. For instance if you have colors defined in a single Float32Array, but need to do vector operations on the elements of the array, instead of copying data out of the Float32Array, wrapArray will give you access to the same data.
Example:
function avarageColor(float32arrayColor){
var sum = vec4.create(),
wrappedArray = vec4.wrapArray(float32arrayColor),
weigth = 1.0/wrappedArray;
for (var i=0;i  < wrappedArray.length;i++){
vec4.add(sum,wrappedArray[i]);
}
return vec4.multiply(sum,[weigth,weigth,weigth,weigth]);
}
Parameters:
array <Float32Array>
