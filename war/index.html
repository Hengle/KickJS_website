KickJS

About

KickJS is a WebGL based game engine build for modern web browsers such as the most recent version of Chrome and Firefox. The source code is Open Source (BSD New License) and the source code is available on GitHub.

Examples

GLSL Shader editor
Video ascii art
Model viewer
Cloth simulation
Snake

Getting started

The best way to get started is to download the library from GitHub and take a look at the files in the tutorial folder. Each of these files contains code that uses a single or a few features of the library. Also take a look at the API documentation.
