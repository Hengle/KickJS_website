Part 0 - Installation

KickJS can be downloaded from (either as zip-file â€“ or you can clone the repository):
https://github.com/mortennobel/KickJS

To use KickJS in a project, you need to use the JavaScript library found in the build folder. Note that the library exists in two versions:

  • kick-min-NN.NN.NN.js: This is the minified version of KickJS. Besides several runtime checks has been disabled to increase runtime performance. This version should be used in production environments.
  • kick-debug-NN.NN.NN.js: This is the uncompressed version of KickJS. It has several built-in checks that logs an error whenever the library is used incorrect. This version should be used during development.

Besides KickJS you need a WebGL capable browser. You can get more info here:
http://get.webgl.org/

To use KickJS you need to include the library in the html-page:

    <html>
    <body>
    <canvas id="3dCanvas" width="500" height="500"></canvas>

    <script type="text/javascript" src="kick-debug-0.4.0.js"></script>
    <script type="text/javascript">
    // init engine (create 3d context)
    var engine = new KICK.core.Engine('3dCanvas');
    </script>
    </body>
    </html>
